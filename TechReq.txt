Веб приложение для систематизации задач.
Основные функции:
1. Создание задач.
2. Просмотр задач.
3. Редактирование задач.
4. Удаление задач.
5. Выполнение задач.
6. Возможно какой-то экран для просмотра статистики (соотноешение выполненных и невыполненных, количество просроченных и тому подобное.)


Экран создания задачи.

Заголовок - Создать задачу.
Поле для ввода - Название задачи.
Поле для ввода даты - Срок выполнения.
Поле для ввода многострочное, растягивается по высоте - Описание задачи.
Кнопка - Отменить.
Кнопка - Создать.

По нажатию на Отменить происходит переход на экран просмотра задач.
По нажатию на Создать происходит создание задачи и переход на просмотр задачи.

Валидация ввода названия задачи - Максимум 50 символов, миниум 1 символ и этот символ не пробел.
Валидация ввода срока выполнения - Не может быть в прошлом.
Валидация ввода описания - Не требуется.


Экран просмотра задачи.

Заголовок - Просмотр задачи.
Текст - Название задачи.
Текст - Описание задачи.
Кнопка - Выполнить.
Кнопка - Вернуть.
Кнопка - Редактировать.
Кнопка - Удалить.
Кнопка - Просмотр задач.

По нажатию на Выполнить - Помечается как выполненая и страница перезагружается.
По нажатию на Вернуть - Задача переходит из выполненого состояния в невыполненное, затем открывется экран редавтирования.
По нажатию на Редактировать - Переход на страницу создания задачи с предзаполненными полями.
По нажатию на Удалить - Удалить задачу, но перед этим запросить подтверждение.
По нажатию на Просмотр задач - Пепреход на просмотр всех задач.
